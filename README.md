# Sign-up Form

This is an exercise to create a sign-up form similar to a provided [design]("https://cdn.statically.io/gh/TheOdinProject/curriculum/5f37d43908ef92499e95a9b90fc3cc291a95014c/html_css/project-sign-up-form/sign-up-form.png").

## Learning Outcome

### HTML
- Creating forms with HTML
- Using different input types, e.g.: e-mail, password etc.
- Basic client-side form validation

### CSS
- Styling form controls
- Pseudo classes and Pseudo elements
